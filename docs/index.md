## 适合谁

LiliaCode 面向需要长期推进工程项目的开发者。它在 Agent 执行层之外提供项目、任务、会话、权限和过程信息的桌面级组织层，让 Claude Code 与 Codex 的协作过程更容易被恢复、追踪和调度。

## 当前状态

LiliaCode 仍处于快速变更阶段，核心功能正在补齐。本地数据库结构可能随新功能调整，数据可能被清空或迁移，请不要在重度生产场景中依赖它保存唯一副本。
